#  CLI Task Manager

Простой менеджер задач для командной строки, написанный на Python.  
Программа позволяет создавать задачи, просматривать их список и отмечать выполненные задачи.

Этот проект сделан как учебный пример CLI-приложения с использованием библиотеки Click и красивого вывода в терминале с помощью Rich.

---

#  Возможности

Программа поддерживает следующие функции:

• добавление новых задач  
• просмотр списка задач  
• отображение статуса задачи (выполнена / не выполнена)  
• отметка задачи как выполненной  
• хранение задач в JSON файле  
• красивый вывод таблицы в терминале  

---

#  Используемые технологии

Проект написан на Python и использует следующие библиотеки:

Python 3 — основной язык программирования  
Click — создание интерфейса командной строки (CLI)  
Rich — форматированный и красивый вывод в терминале  
JSON — хранение данных задач  
OS — работа с файловой системой

#  Установка

1. Установите Python 3.

2. Установите необходимые библиотеки:

pip install click rich

3. Склонируйте репозиторий:

git clone https://github.com/your-username/task-manager-cli.git

4. Перейдите в папку проекта:

cd task-manager-cli

#  Как работает программа

Все задачи сохраняются в файле tasks.json.

Каждая задача имеет два поля:

title — название задачи  
done — статус выполнения (true или false)

Пример задачи:

{
  "title": "Купить молоко",
  "done": false
}

Если задача выполнена:

{
  "title": "Сделать проект",
  "done": true
}

---

#  Использование программы

## Показать список задач

Команда:

python main.py list

Если задач нет, программа покажет сообщение:

Задач нет

Если задачи есть, будет выведена таблица:

ID | Задача | Статус

Статус отображается так:

⏳ — задача не выполнена  
✅ — задача выполнена

---

## Добавить новую задачу

Команда:

python main.py add "Купить молоко"

После добавления появится сообщение:

✅ Добавлена задача: Купить молоко

Задача автоматически сохраняется в файл tasks.json.

---

## Отметить задачу выполненной

Команда:

python main.py complete 1

Где число — это ID задачи.

После выполнения появится сообщение:

✅ Задача 1 отмечена выполненной

---

#  Хранение данных

Все задачи сохраняются в JSON формате.

Пример файла tasks.json:

[
  {
    "title": "Купить молоко",
    "done": false
  },
  {
    "title": "Сделать проект",
    "done": true
  }
]
